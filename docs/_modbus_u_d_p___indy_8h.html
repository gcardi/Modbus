Detailed Description

Modbus::Master::UDPProtocolIndy — Embarcadero Indy TIdUDPClient transport for Modbus UDP.

Provides a Modbus UDP master implementation using the Embarcadero Indy TIdUDPClient component. Response datagrams are cached in an internal buffer; DoRead() slices bytes from the cache so that TCPIPProtocol's multi-step read logic works transparently with datagram sockets.
