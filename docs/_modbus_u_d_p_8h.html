Detailed Description

Modbus::Master::UDPProtocol — UDP marker class in the MBAP protocol hierarchy.

UDPProtocol is a thin intermediate class that identifies UDP-based transports in the class hierarchy without adding behaviour. Concrete implementations (UDPProtocolWinSock, UDPProtocolIndy) derive from this class and provide the actual UDP socket I/O.
