Detailed Description

Serial port enumeration utilities using the Windows SetupAPI.

Provides SvcApp::Utils::EnumSerialPort(), a function template that enumerates all COM ports present in the system by querying the Windows SetupAPI device information set for GUID_DEVINTERFACE_COMPORT. For each port found, a caller-supplied functor is invoked with the port name and device description, and its return value is appended to the output iterator.

The default functor, BuildSerialPortInfoTupleFnctr, packages the name and description into a boost::tuple<String,String>.
