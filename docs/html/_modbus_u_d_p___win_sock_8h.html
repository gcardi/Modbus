Detailed Description

Modbus::Master::UDPProtocolWinSock — WinSock2 UDP transport for Modbus UDP.

Provides a Modbus UDP master implementation built on the Windows WinSock2 API. Key characteristics:

  • WSAStartup / WSACleanup called in constructor / destructor.
  • Hostname resolution via GetAddrInfoW() (supports Unicode hostnames).
  • DoWrite() sends a complete MBAP datagram via sendto(), then immediately receives the server response via recvfrom() and stores it in an internal receive buffer.
  • DoRead() slices the requested number of bytes from the cached receive buffer.
  • SO_RCVTIMEO set to 2 seconds for the recvfrom() call.
