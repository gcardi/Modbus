Detailed Description

Modbus::Master::TCPProtocol — TCP marker class in the MBAP protocol hierarchy.

TCPProtocol is a thin intermediate class that identifies TCP-based transports in the class hierarchy without adding behaviour. Concrete implementations (TCPProtocolWinSock, TCPProtocolIndy) derive from this class and provide the actual TCP socket I/O.
