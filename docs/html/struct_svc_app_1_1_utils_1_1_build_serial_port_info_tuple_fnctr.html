Detailed Description

Default functor for EnumSerialPort that packages the port name and description into a boost::tuple.

When passed as the Fn argument to EnumSerialPort(), each discovered port is represented as a boost::tuple<String,String> where the first element is the port name (e.g., "COM3") and the second element is the human-readable device description.
