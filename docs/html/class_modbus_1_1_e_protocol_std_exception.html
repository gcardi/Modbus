Detailed Description

template<ExceptionCode Code>
class Modbus::EProtocolStdException< Code >

Typed Modbus protocol exception parameterised on a specific ExceptionCode.

This template is instantiated once for each standard Modbus exception code and then aliased to a named exception type (EIllegalFunction, EIllegalDataAddress, etc.). This allows catch blocks to distinguish exception codes without inspecting GetCode().

Template Parameters
CodeThe specific Modbus ExceptionCode this exception represents.
