Detailed Description

Modbus::Master::TCPProtocolIndy — Embarcadero Indy TIdTCPClient transport for Modbus TCP.

Provides a Modbus TCP master implementation using the Embarcadero Indy (Internet Direct) TIdTCPClient component. This is the preferred transport when building C++Builder / RAD Studio VCL applications that already use the Indy library.
