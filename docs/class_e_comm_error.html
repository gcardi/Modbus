Detailed Description

Exception thrown by TCommPort when a Win32 serial API call fails.

Inspect the public Error member to determine which operation failed, and Errno (GetLastError() value) for the underlying OS error code. Errno can be passed to FormatMessage() to obtain a human-readable description.
