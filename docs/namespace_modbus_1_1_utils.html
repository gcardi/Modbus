template<typename OutputIterator, typename TF>
void Modbus::Utils::EnumSerialPort ( OutputIterator Out,
TF Fn = BuildSerialPortInfoTupleFnctr() )

Enumerates all COM ports currently present in the system.

Uses the Windows SetupAPI (SetupDiGetClassDevs with GUID_DEVINTERFACE_COMPORT) to iterate over installed COM port devices. For each device that has a valid "PortName" registry value, the functor Fn is called with the port name and device description, and the return value is written to the output iterator Out.

Template Parameters
OutputIteratorOutput iterator type whose value type matches the return type of Fn.
TFFunctor type callable as Fn(String portName, String description).
Parameters
OutOutput iterator that receives one entry per discovered COM port.
FnFunctor invoked for each port (default: BuildSerialPortInfoTupleFnctr).
Exceptions
std::runtime_error(via RaiseLastOSError) if SetupDiGetClassDevs fails.
Example
std::vector<boost::tuple<String,String>> ports;
Modbus::Utils::EnumSerialPort( std::back_inserter(ports) );
for ( auto& p : ports )
ShowMessage( p.get<0>() + " — " + p.get<1>() );
