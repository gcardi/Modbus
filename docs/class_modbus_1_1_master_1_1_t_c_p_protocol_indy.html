Detailed Description

Embarcadero Indy TIdTCPClient-based TCP implementation of the Modbus master protocol.

Uses a TIdTCPClient instance (managed via std::unique_ptr) for the TCP connection. Indy handles hostname resolution, connection management, and blocking I/O internally, making this implementation straightforward and VCL-friendly.

Note
Requires the Embarcadero Indy library (IndyCore, IndyProtocols, IndySystem). This transport is only available in C++Builder / RAD Studio environments.
